# Rux Art

This repository contains source and exported artwork for the Rux programming language.

## Folders

- `Src` contains the original editable artwork files made by [Affinity](https://www.affinity.studio/).
- `Dist` contains exported assets ready for use in websites, documentation, applications, and social media.
